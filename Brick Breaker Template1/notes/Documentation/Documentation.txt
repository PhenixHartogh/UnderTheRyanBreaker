This note is high-level documentation for the core parts of this template project.
All events in this project are commented, so you can open any event and read the comments to understand what it does, and how it works.


# FONTS

There is 1 font in this project

fnt_font
	Used for any text drawn

# OBJECTS

There are 36 objects in this project

obj_base_bound
	Game object that acts as a parent for vertical and horizontal bounds to help calculate collisions with boundaries
	
obj_bound_hori
	Game object: A horizontal child of obj_base_bound

obj_bound_vert
	Game object:A vertical child of obj_base_bound

obj_base_brick
	Game object to give bricks their basic functionality
	
	Create							Sets up basic variables
	Destroy							Plays a sound effect for destruction
	User Event 0					Decrease hitpoints and have a chance to spawn a powerup

obj_brick_blue
	Game object: A child object of obj_base_brick with the same functionality but 1hp and worth 100 points

obj_brick_green
	Game object: A child object of obj_base_brick with the same functionality but 3hp and worth 300 points

obj_brick_indestructible
	Game object collider that cannot be destroyed

obj_brick_pink
	Game object: A child object of obj_base_brick with the same functionality but 2hp and worth 200 points

obj_brick_yellow
	Game object: A child object of obj_base_brick with the same functionality but 4hp and worth 400 points

obj_hit_0
	Game object sprite effect instantiated when a brick is destroyed

obj_hit_1
	Game object sprite effect instantiated when a brick is hit

obj_hit_base
	Game object sprite effect instantiated when a ball and brick collide
	
	Animation End					Destroys the effect

obj_popup
	Game object popup to show that the player has collected a points powerup
	
	Create							Set basic variables
	Alarm 0							Destroy the popup
	Draw GUI						Draw the points message

obj_button_base
	UI object basic button with hover and press functionality
	
	Draw GUI End					Draws the button at the latest possible point
	Left Pressed					Plays a click sound
	Mouse Enter						Changes the draw tint and opacity
	Mouse Leave						Changes the draw tint and opacity back to normal

obj_button_menu
	UI object child of obj_button_base that moves the game back to the main menu - blue version
	
	Left Pressed					Moves the game back to the main menu and stops all music

obj_button_menupurple
	UI object child of obj_button_base that moves the game back to the main menu - purple version
	
	Left Pressed					Moves the game back to the main menu and stops all music

obj_button_menured
	UI object child of obj_button_base that moves the game back to the main menu - red version
	
	Left Pressed					Moves the game back to the main menu and stops all music

obj_button_next
	UI object child of obj_button_base that moves the game to the next level
	
	Left Pressed					Moves the game to the next level

obj_button_play
	UI object child of obj_button_base that moves the game to the first level
	
	Left Pressed					Moves the game to the first level

obj_button_quit
	UI object child of obj_button_base that quits the game
	
	Left Pressed					Quit the game

obj_button_restart
	UI object child of obj_button_base that restarts the level
	
	Left Pressed					Restart the room

obj_button_resume
	UI object child of obj_button_base that resumes the game
	
	Left Pressed					Destroy any menu object that exists then destroy itself

obj_button_retry
	UI object child of obj_button_base that restarts the level
	
	Left Pressed					Restart the room

obj_menu_base
	UI object that acts as a base for all other menu objects
	
	Create							Set alarm 0
	Alarm 0							Show the cursor

obj_menu_lose
	UI object child of obj_menu_base displayed when the game is lost

obj_menu_pause
	UI object child of obj_menu_base displayed when the game is paused
	
	Destroy							Hide the cursor

obj_menu_win
	UI object child of obj_menu_base displayed when the game is won

obj_powerup_base
	Game object parent of all other powerups
	
	Create							Set its vertical speed and play a spawn noise
	Step							Pause vertical movement if the game is paused
	-><- obj_bat					Destroy itself and play a powerup noise

obj_powerup_extralife
	Game object that gives the player an extra life when they hit it with the bat
	
	-><- obj_bat					Give the player an extra life

obj_powerup_largebat
	Game object that makes the player's bat larger when it is hit
	
	-><- obj_bat					Set the large bat timer to 4 seconds assuming 60fps

obj_powerup_multiball
	Game object that spawns another ball when it is hit
	
	-><- obj_bat					Spawn another ball

obj_powerup_pointball
	Game object that grants more points
	
	-><- obj_bat					Grant the player 1000 points and create a popup to display this

obj_ball
	Game object that bounces around the screen and off the bat, hitting blocks and granting points
	
	Create							Set the ball moving in some direction but lock it to the bat if it's the only one
	Step							Keep the ball locked to the bat if it should be, and pause its movement while the game is paused
	Draw							Draw the ball and a trail behind it
	-><- obj_base_bound				Bounce the ball off the boundary
	-><- obj_base_brick				Bounce the ball off the brick and decrease its health
	-><- obj_bat					Bounce the ball off the bat
	-><- obj_brick_indestructible	Bounce the ball off the brick but don't decrease health

obj_bat
	Game object 
	
	Create							Set up powerup variables (long bat, flash timer)
	Step							Move the bat to match the mouse and work out powerup times
	Alarm 0							Change the image speed to make the bat flash slowly
	Key Down - Left					Move the bat left
	Key Down - Right				Move the bat right
	Animation End					Pause the animation and randomise when the bat will next flash

obj_main
	Game object that handles overarching game variables (score, paused, cursor)
	
	Create							Set up the overarching variables
	Step							Handle life loss, level management
	Draw GUI						Draw the score and lives
	Key Down - Escape				Pause the game

obj_menu_music
	Sound object that plays the main menu music
	
	Create							Play the main menu music

# ROOMS

There are 4 rooms in this project

rm_menu
	Main menu room
	
rm_level_1
	The first (and easiest) level game room
	
	Assets_1						Asset layer for the game boundaries and pause direction
	Effects							Instance layer for brick collision effects
	Sequences						Asset layer for sequence menus
	Instances						Instance layer for various game objects such as bricks, the bat, balls, powerups
	Background						Background layer so the game is not in a void
	
rm_level_2

	Effects							Instance layer for brick collision effects
	Sequences						Asset layer for sequence menus
	Instances						Instance layer for various game objects such as bricks, the bat, balls, powerups
	Background						Background layer so the game is not in a void
	
rm_level_3
	The third (and hardest) level game room
	
	Assets_1						Asset layer for the game boundaries and pause direction
	Effects							Instance layer for brick collision effects
	Sequences						Asset layer for sequence menus
	Instances						Instance layer for various game objects such as bricks, the bat, balls, powerups
	Background						Background layer so the game is not in a void

# SEQUENCES

There are 4 sequences in this project

seq_menu_end_game
	Menu played at the end of the game that forces the player back to the main menu

seq_menu_end_level
	Menu played at the end of a level

seq_menu_lose
	Menu played if a player loses in a level
	
seq_menu_paused
	Menu played when the player pauses the game

# SOUNDS

There are 10 sound objects in this project

They are organised by their type
	Music							Played in the background (often looped)
	SFX								Called when needed in game

# SPRITES

There 44 sprites in this project

They are organised by their use
	Backgrounds						Background sprites for the background of rooms
	Boundary						Sprites for boundary blocks
	Bricks							All bricks with each stage of hp needed
	InGame							Holds all dynamic gameplay sprites like the bat and ball
	Menus							Holds backgrounds and buttons for menus
	Powerups						All powerup sprites
	UI								Sprites for dynamic in-game UI like score and lives
